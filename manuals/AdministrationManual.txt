Update Ubuntu
	$ sudo apt-get install
	$ sudo apt-get upgrade
Create user with sudo rigths - 
	$ useradd username -m #Creat user and a home dir
	$ usermod -g sudo username #Change users primary group to sudo
Create a password for user:
 	$ sudo passwd username
Install git
	$ sudo apt-get install git

Install mvn - {required}
	$ sudo apt-get install maven

Navigate to http://heroku.com and create an account.	
Heroku toolbelt 
	Install:
	$ wget -qO- https://toolbelt.heroku.com/install-ubuntu.sh | sh

		Once installed, you'll have access to the heroku command from your command shell. Log in using the email address and password you used when creating your Heroku account:
	
	$ heroku login
		Enter your Heroku credentials.
		Email: adam@example.com
		Password:
		Could not find an existing public key.
		Would you like to generate one? [Yn]
		Generating new SSH public key.
		Uploading ssh public key /Users/adam/.ssh/id_rsa.pub

		You're now ready to create your first Heroku app:

	$ cd ~/myapp  #Git hub forked root 
	$ heroku create
		Creating stark-fog-398... done, stack is cedar
		http://stark-fog-398.herokuapp.com/ | git@heroku.com:stark-fog-398.git
		Git remote heroku added

Install gem - {optional - Travis requirement}
	Install ruby1.9.1-dev:
		$ sudo apt-get update
		$ sudo apt-get install -y make ruby1.9.1-dev

Travis tool - {optional}
	Install:
	Use gem1.9.1 to install travis:

		$ sudo gem1.9.1 install travis

Make a hook to Travis-CI on Github
	Log in to GitHub.
	Navigate to the new repository.
	Navigate to Settings.
	Navigate to Service Hooks.
	Navigate to Travis and follow instructions on GitHub page.

	You have now set up a hook to your new GitHub repository.

Make changes to .travis.yml for your Heroku Apps.

	First of all, you have to get your auth token from Heroku :
	
		$token=$(heroku auth:token)

	This will store the token, and keep it for later use.

	Then, install the Travis CLI tool:

		$sudo gem install travis

	You now have the Travis CLI tool installed. This will help you to encrypt your Heroku token.
	
	travis encrypt HEROKU_API_KEY=$token --add
	
	Running the above command will add your encrypted token to your .travis.yml.

	Make changes to your .travis.yml for your heroku apps.

Do some changes and push to github and it will deploy to your production server.
